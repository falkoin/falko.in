Changing handbrake's automatic aspect ratio detection

3 March 2021 One-minute read
solutions
handbrake • video

From time to time I do screen recordings, which include a PowerPoint with an aspect ratio of 4:3. When encoding these video files with handbrake it tries to be smart about it and automatically detects the aspect ratio of the video, which usually results in a cropped video.
But there is an easy fix for that. Go to the picture tab and change the cropping option to 0 in all directions.
